Project 1

FizzBuzz is a basic programming exercise that prints numbers in a sequence while applying simple rules. For each number, if it is divisible by 3, the program outputs "Fizz"; if it is divisible by 5, it outputs "Buzz"; and if it is divisible by both 3 and 5, it outputs "FizzBuzz." Otherwise, the program just prints the number itself.
