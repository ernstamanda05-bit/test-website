Project 1

FizzBuzz is a basic programming exercise that prints numbers in a sequence while applying simple rules. For each number, if it is divisible by 3, the program outputs "Fizz"; if it is divisible by 5, it outputs "Buzz"; and if it is divisible by both 3 and 5, it outputs "FizzBuzz." Otherwise, the program just prints the number itself.

Project 2

MonsterBattle is a Java-based program that simulates a battle between a hero and multiple monsters. The program reads monster data from files, creates monster objects, and runs a battle sequence where the hero fights each monster one by one. Throughout the simulation, the program tracks outcomes such as defeated monsters and the hero’s remaining health, and displays a final summary at the end. This project demonstrates the use of object-oriented programming, file input using scanners, exception handling, and control structures such as loops and conditionals.
